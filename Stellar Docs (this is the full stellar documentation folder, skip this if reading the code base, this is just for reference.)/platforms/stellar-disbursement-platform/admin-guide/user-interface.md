# User Interface

A description and walkthrough of the various parts of the SDP user interface.

[## 챨혺혫혙챦쨍혧 Dashboard Home

The main page of the dashboard contains a summary of recent disbursement activity and key performance metrics.](/docs/platforms/stellar-disbursement-platform/admin-guide/user-interface/dashboard-home.md)

[## 챨혺혫혙챦쨍혧 Disbursements

The Disbursements page provides a paginated list of all disbursements, detailing each disbursement's status and related payment information.](/docs/platforms/stellar-disbursement-platform/admin-guide/user-interface/disbursements.md)

[## 챨혺혫혙챦쨍혧 Receivers

The Receivers page displays a list of individuals set to receive payments, with wallet information and payment history. This information allows you to track and manage the payments made to each receiver, and provides a snapshot of each receiver's interaction with a disbursement.](/docs/platforms/stellar-disbursement-platform/admin-guide/user-interface/receivers.md)

[## 챨혺혫혙챦쨍혧 Payments

The Payments page provides a list of all payments, detailing each payment's status and related information.](/docs/platforms/stellar-disbursement-platform/admin-guide/user-interface/payments.md)

[## 챨혺혫혙챦쨍혧 Wallets

The Wallets page provides detailed information about your distribution account, which is the primary Stellar account from which your disbursements are made.](/docs/platforms/stellar-disbursement-platform/admin-guide/user-interface/wallets.md)

[## 챨혺혫혙챦쨍혧 Analytics

The Analytics page provides comprehensive insights into various aspects of financial transactions, enabling the user to track and understand key payment-related metrics. As more metrics and statistics become available, additional tiles will be added to this screen. The page displays information such as the successful payment rate, the total number of successful payments, the number of failed payments, and the number of remaining payments. Additionally, it shows the total amount disbursed, the average amount per transaction, the total amount in USDC, and the number of individuals and wallets involved in the transactions.](/docs/platforms/stellar-disbursement-platform/admin-guide/user-interface/analytics.md)

[## 챨혺혫혙챦쨍혧 Circle Configuration

If the tenant was created with a [Circle] distribution account, then the tenant owner will need to manually configure that account from within the SDP dashboard.](/docs/platforms/stellar-disbursement-platform/admin-guide/user-interface/circle-configuration.md)
